자주 묻는 질문

계약 전에 꼭 확인해주세요.

구축까지 얼마나 걸리나요? +
STANDARD 5일, DELUXE 10일, PREMIUM 15일이 기준입니다. 복잡도나 커스터마이징 범위에 따라 조정될 수 있고, 사전 상담 때 정확한 일정 안내드립니다.
월 운영비는 정확히 얼마인가요? +
초기 세팅비(1회) 외에 알림톡 건당 13원, AI 채점 API 실사용량만큼 과금됩니다. 학생 30명 기준 월 1~3만원, 100명 기준 월 5~8만원 수준이에요. 비싼 월정액 프로그램 없이 쓰신 만큼만 내시면 됩니다.
기존 엑셀 · Notion 데이터는 어떻게 하나요? +
쓰시던 데이터 그대로 가져와서 자동화 시스템에 연결해드립니다. Notion은 양방향 동기화 가능하고, 엑셀은 일괄 업로드로 한 번에 이관돼요.
구축 후에도 수정할 수 있나요? +
수정 횟수는 제한이 없습니다. 납품 후에도 학원 운영하면서 "이 부분 좀 바꿔주세요"라고 하시면 대응해드려요. PREMIUM은 30일 집중 A/S 기간이 포함되고, 그 이후도 합리적인 선에서 계속 지원합니다.
세금계산서 발행되나요? +
현재 개인회원으로 운영 중이라 세금계산서 발행은 불가합니다. 크몽 결제 영수증으로 대체 가능하며, 법인 거래가 꼭 필요하신 경우 별도 문의 부탁드립니다.
학원에 상주해서 작업하시나요? +
상주는 어렵습니다. 대신 카톡·전화·화상 미팅으로 밀착 소통하고, 구축 완료 후에도 언제든 연락 가능합니다 (평균 응답 1시간 이내).
IB 수학 외 다른 과목도 가능한가요? +
네. 출결·수업료·알림 자동화는 모든 과목·학년에 공통 적용됩니다. 자동 채점만 과목별 적합성이 다른데, 사전 상담 때 "우리 학원 시험 유형"을 보고 정확히 안내드립니다.
